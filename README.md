
# This is a Timer App created with React.

## Features
- Set the time
- Start the timer
- Stop the timer
- Reset the timer
- Pause the timer
- Resume the timer

## Installation
```bash
    git clone https://github.com/aviraltiwari/Timer-App
    npm install
```

## Usage
```bash
    npm start
```
